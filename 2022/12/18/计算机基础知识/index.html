    • 字
    • 位
    • 字节
    • 字长
    • ✔️ 计算机处理数据时，CPU 通过数据总线一次存取，加工和传送的数据称为 字。
    • ❌
    • ❌
    • ❌ 计算机的运算部件能同时处理的二进制数据的位数成为字长

  2. 在计算机中，如果一个存储单元能存放一个字节，则容量为 32KB 的存储器中的存储单元个数为 。

    • 32768
    • 32000
    • 16384
    • 65536
    • ✔️ 32KB = 32 x 1024B = 32768B

  3. 汉字的字形码是用来解决汉字的 问题。

    • 输入
    • 存储
    • 输出
    • 计算
    • ❌
    • ❌
    • ✔️ 汉字字形码是记录汉字外形，是汉字的输出形式
    • ❌

  4. 下列说法正确的是 。

    • 同一个汉字的输入码的长度随输入方法的不同而不同
    • 一个汉字的机内码与它的国际码是相同的
    • 不同汉字的机内码的长度是不相同的
    • 同一汉字用不同的输入法输入时，其机内码是不相同的

  5. 在计算机中，一个字节不能表示 问题。

    • 任意 3 位八进制数
    • 2 位十六进制数
    • 一个 ASCII 码字符
    • 256 种状态

  6. 编译程序的作用是 。

    • 执行源程序
    • 管理软件资源的使用
    • 将源程序转换为机器语言程序
    • 管理硬件资源的使用
    • ❌
    • ❌
    • ✔️ 编译程序将用高级语言编写的源程序翻译成与之等价的用机器语言表示的目的程序，其翻译过程称为编译
    • ❌

  7. 下列叙述中，正确的是 。

    • 计算机能直接识别并执行用高级程序设计语言编写的程序
    • 用机器语言编写的程序可读性最差
    • 机器语言就是汇编语言
    • 高级语言的编译系统是应用软件
    • ❌ 计算机能直接识别并执行用 机器语言 编写的程序
    • ✔️
    • ❌ 机器语言和汇编语言都属于低级语言，但是两者不同
    • ❌ 高级语言的编译系统是 系统软件

  8. 指令的操作码表示的是 。

    • 做什么操作
    • 停止操作
    • 操作结果
    • 操作地址
    • ✔️ 操作码规定操作的类型，，即进行什么样的操作，地址码规定了要操作的数据存放位置，以及操作结果存放的地址。
    • ❌
    • ❌
    • ❌

  9. 计算机之所以能按人们的意愿自动进行工作，主要是因为采用了 。

    • 二进制存储数制
    • 高速电子元器件
    • 存储程序控制
    • 程序设计语言
    • ❌
    • ❌
    • ✔️ 计算机能够自动完成运算或处理过程的基础是 存储程序 工作原理
    • ❌

  10. 下列有关存储器的说法中，错误的是 。

    • 存储器的存储单元是存储单元，每个单元都有一个唯一的地址
    • 存储器的主要功能是用于存放程序和数据
    • 内存也成为工作存储器，是 CPU 可以直接访问的存储器
    • 当前运行的程序和数据都必须放在 ROM 中，否则无法调用
    • ❌
    • ❌
    • ❌
    • ✔️ 当前运行的程序和数据都必须放在 RAM 中，否则无法调用

  11. 下列关于运算器的说法中，错误的是 。

    • 运算器是计算机的加工处理部件
    • 运算器的运算功能是由一个称为算数逻辑运算单元 (简称 ALU) 的数字电路实现的
    • ALU 的运算操作包括算数运算和逻辑运算
    • ALU 具有记忆保存功能
    • ❌
    • ❌
    • ❌
    • ✔️ ALU 本身是一个没有内部记忆功能的组合逻辑电路。

  12. 计算机中常用 MIPS (Million Instruction Per Second) 来描述 。

    • 计算机的运算速度
    • 计算机的可靠性
    • 计算机的可扩充性
    • 计算机的可运行性
    • ✔️ 计算机的运算速度用 CPU 每秒能执行的指令条数来表示，单位一般为 MIPS (每秒百万条指令)
    • ❌
    • ❌
    • ❌

  13. 在以下打印机中，打印速度最快的是 。

    • 针式打印机
    • 激光打印机
    • 点阵打印机
    • 喷墨打印机
    • ❌
    • ✔️ 激光打印机打印速度快，质量高，成本低。
    • ❌
    • ❌

  14. CPU 中正在执行的指令存放于 。

    • 指令寄存器
    • 指令计数器
    • 指令译码器
    • 指令控制器
    • ✔️ 指令寄存器是用来保存当前正在执行的指令对象通过译码器解释该指令的含义，控制运算器的操作，记录 CPU 的内部状态
    • ❌ 指令计数器是用来存放 CPU 正在执行的指令的地址，CPU 将按照该地址从内存读取所需要执行的指令
    • ❌
    • ❌

  15. 决定 CPU 寻址能力的是 。

    • 控制总线的位数
    • 地址总线的位数
    • 数据总线的位数
    • 内存的大小

  16. 下列关于 BIOS 的叙述，错误是 。

    • BIOS 是一组机器语言程序，又称作基本输入 / 输出系统
    • 操作系统启动后，CPU 会执行 BIOS 中的程序
    • BIOS 具有诊断计算机故障的功能
    • BIOS 程序存放在 ROM 中，断电后不会使其内容丢失
    • ❌
    • ✔️ 操作系统 启动中 CPU 会执行 BIOS 中的程序
    • ❌
    • ❌

  17. 光盘上记录信息的光道是由里向外的螺旋线，磁盘上记录信息的磁道是 。

    • 同心圆
    • 椭圆
    • 等高线
    • 由里向外的螺旋线
    • ✔️ 光盘使用一条光道来记录数据，而磁盘是将数据记录在磁道上，由一圈一圈的同心圆构成，通常磁盘有 80 个磁道，每个磁道有 18 个扇区。
    • ❌
    • ❌
    • ❌

  18. 决定 CPU 芯片内部数据传输与操作速度的最主要因素是 。

    • Cache 容量
    • 指令系统
    • 主频
    • 运算器结构
    • ❌
    • ❌
    • ✔️ 一般而言，主频越高，执行一条指令需要的时间就较短，CPU 的处理速度就越快。
    • ❌

  19. 根据存储器芯片的功能及物理特性，目前用作 PC 机主存储器的是 。

    • DRAM
    • EPROM
    • Flash ROM
    • SRAM
    • ❌ 主存储器主要是由 DRAM 芯片组成的，它包含有大量的存储单元，每个存储单元都可以存放一个字节。
