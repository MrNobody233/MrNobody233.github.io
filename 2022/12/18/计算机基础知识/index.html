# 单选题

  1. 计算机处理数据时，CPU 通过数据总线一次存取，加工，和传送的数据称为 。
    • 字
    • 位
    • 字节
    • 字长

计算机处理数据时，CPU 通过数据总线一次存取，加工和传送的数据成为 字。 计算机的运算部件能同时处理的二进制数据的位数成为字长

  1. 在计算机中，如果一个存储单元能存放一个字节，则容量为 32KB 的存储器中的存储单元个数为 。
    • 32768
    • 32000
    • 16384
    • 65536

32KB = 32 x 1024B = 32768B

  1. 汉字的字形码是用来解决汉字的 问题。
    • 输入
    • 存储
    • 输出
    • 计算

汉字字形码是记录汉字外形，是汉字的输出形式

  1. 下列说法正确的是 。

    • 同一个汉字的输入码的长度随输入方法的不同而不同
    • 一个汉字的机内码与它的国际码是相同的
    • 不同汉字的机内码的长度是不相同的
    • 同一汉字用不同的输入法输入时，其机内码是不相同的

  2. 在计算机中，一个字节不能表示 问题。

    • 任意 3 位八进制数
    • 2 位十六进制数
    • 一个 ASCII 码字符
    • 256 种状态
