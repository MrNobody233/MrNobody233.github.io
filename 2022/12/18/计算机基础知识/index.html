    • 同一个汉字的输入码的长度随输入方法的不同而不同

    • 不同汉字的机内码的长度是不相同的

    • ❌

  7. 下列叙述中，正确的是 。

    • 计算机能直接识别并执行用高级程序设计语言编写的程序
    • 用机器语言编写的程序可读性最差
    • 机器语言就是汇编语言
    • 高级语言的编译系统是应用软件
    • ❌ 计算机能直接识别并执行用 机器语言 编写的程序
    • ✔️
    • ❌ 机器语言和汇编语言都属于低级语言，但是两者不同
    • ❌ 高级语言的编译系统是 系统软件

    • 做什么操作

    • 操作地址

    • 存储器的存储单位是存储单元，每个单元都有一个唯一的地址
    • 存储器的主要功能是用于存放程序和数据
    • 内存也成为工作存储器，是 CPU 可以直接访问的存储器
    • 当前运行的程序和数据都必须放在 ROM 中，否则无法调用

  11. 下列关于运算器的说法中，错误的是 。

    • 运算器是计算机的加工处理部件
    • 运算器的运算功能是由一个称为算数逻辑运算单元 (简称 ALU) 的数字电路实现的
    • ALU 的运算操作包括算数运算和逻辑运算

  12. 计算机中常用 MIPS (Million Instruction Per Second) 来描述 。

    • 计算机的运算速度
    • 计算机的可靠性
    • 计算机的可扩充性
    • 计算机的可运行性
    • ✔️ 计算机的运算速度用 CPU 每秒能执行的指令条数来表示，单位一般为 MIPS (每秒百万条指令)

    • ❌

    • 指令寄存器
    • 指令计数器
    • 指令译码器
    • 指令控制器
    • ✔️ 指令寄存器是用来保存当前正在执行的指令对象通过译码器解释该指令的含义，控制运算器的操作，记录 CPU 的内部状态
    • ❌ 指令计数器是用来存放 CPU 正在执行的指令的地址，CPU 将按照该地址从内存读取所需要执行的指令

    • 控制总线的位数
    • 地址总线的位数
    • 数据总线的位数
    • 内存的大小

  16. 下列关于 BIOS 的叙述，错误是 。

    • BIOS 是一组机器语言程序，又称作基本输入 / 输出系统
    • 操作系统启动后，CPU 会执行 BIOS 中的程序
    • BIOS 具有诊断计算机故障的功能
    • BIOS 程序存放在 ROM 中，断电后不会使其内容丢失
    • ❌
    • ✔️ 操作系统 启动中 CPU 会执行 BIOS 中的程序
    • ❌
    • ❌

  17. 光盘上记录信息的光道是由里向外的螺旋线，磁盘上记录信息的磁道是 。

    • 同心圆
    • 椭圆
    • 等高线
    • 由里向外的螺旋线
    • ✔️ 光盘使用一条光道来记录数据，而磁盘是将数据记录在磁道上，由一圈一圈的同心圆构成，通常磁盘有 80 个磁道，每个磁道有 18 个扇区。

    • ❌
    • ❌

    • Cache 容量

    • 主频
    • 运算器结构
    • ❌
    • ❌
    • ✔️ 一般而言，主频越高，执行一条指令需要的时间就较短，CPU 的处理速度就越快。
    • ❌

    • DRAM
    • EPROM
    • Flash ROM
    • SRAM
    • ✔️ 主存储器主要是由 DRAM 芯片组成的，它包含有大量的存储单元，每个存储单元都可以存放一个字节。

    • ❌
    • ❌

  20. 下列关于 PC 机主板的叙述，错误的是 。

    • BIOS 的中文名称是基本输入输出系统，它仅包含基本外围设备的驱动程序
    • CMOS 由电池供电，当电池无电时，其中设置的信息会丢失
    • Cache 通常是指由 SRAM 组成的高速缓冲存储器
    • 目前 PC 机主板上的芯片一般由多块 VLSI 组成，不同类型的 CPU 通常需要使用不同的芯片组
    • ✔️ BIOS 不仅包含基本外围设备的驱动程序，还包含 POST (加电自检) 程序，系统自举 (装入) 程序，启动 CMOS 设置程序和 CMOS 设置程序
    • ❌
    • ❌
    • ❌

  21. 下列关于光盘存储器的叙述，错误的是 。

    • CD 的存储容量小于 DVD 的存储容量
    • CD-R 是一种只读不能写的光盘存储器。
    • CD-RW 是一种既能读又能写的光盘存储器。
    • DVD 光盘也能读取 CD 光盘上的数据

    • ✔️ CD-R 光盘是一种只读但只能够写一次的光盘

  22. 下列关于 I/O 设备的叙述，错误的是 。

    • I/O 设备的操作由 CPU 启动
    • I/O 设备的操作在 I/O 控制器下完成

    • 同一时刻只有一个 I/O 设备工作

  23. 下列关于 CPU 的叙述，错误的是 。

    • CPU 中指令计数器的作用是统计已经执行过的指令数目
    • CPU 所能执行的全部指令的集合称为改 CPU 的指令系统
    • CPU 中含有诺干个寄存器
    • 时钟频率决定着 CPU 芯片内部数据传输与操作速度的快慢
    • ✔️ CPU 中指令计数器的作用是存放下一条执行指令所在单元地址的地方，并不是统计已经执行过的指令数目。

    • ❌
    • ❌

    • CPU 数据线的宽度
    • 主板芯片组的型号
    • 主板存储器插座类型与数目
    • CPU 地址线的宽度
    • ✔️ 数据线的宽度大小影响 CPU 当前可以传输数据量的大小。

# 多选题

  1. 衡量微机的性能所采用的的技术指标有

    • 显示器分辨率
    • 主频
    • 体积
    • 存取周期
    • ❌
    • ✔️ 微机的主要性能指标有： 主频，字长，存取周期，内存容量等
    • ❌
    • ✔️

  2. 微型计算机的系统总线是 CPU 与其他部件之间传输和数据地址等信息的公共通道

    • 输出
    • 输入
    • 控制信号
    • 数据
    • ❌
    • ❌
    • ✔️ 微机中的总线一般分为数据总线，地址总线和控制总线，分别用来传输数据，数据地址以及控制信号。
    • ✔️

  3. 关于二进制的说法正确的是

    • 运算法则为借一当二
    • 二进制数各位的位权是以二为底的幂
    • 二进制的奇数是 2
    • 它由 1,2 这两个数码组成
    • ✔️
    • ✔️
    • ✔️
    • ❌二进制数由 0 和 1 两个数码组成。

  4. 下列关于汉字编码的叙述正确的是

    • 在不同的汉字输入法中，同一个汉字的输入码通常不同
    • 在 GB2312 中，汉字的国际码为该汉字的区号和位号分别加 32 之后得到的二进制代码
    • 在 GB2312 中，汉字内码和每个字节的最高位是 0 或 1
    • 在不同字体 (宋体，仿宋体，楷体等) 的字形描述信息存放在同一个字库中
    • ✔️
    • ✔️
    • ❌ 汉字内码的每个字节的最高位是 1
    • ❌ 不同字体的字形描述信息存放在不同字库中

  5. 下列部件，组成中央处理器的是

    • 硬盘
    • 控制器
    • 运算器
    • 存储器
    • ❌
    • ✔️ 控制器和运算器合称为中央处理器
    • ✔️
    • ❌

  6. 关于计算机内存，下列说法正确的是

    • CPU 从 RAM 中既可以读出信息又可以写入信息
    • 1MB 内存通常是指 1024x1024 字节大小的内存
    • 计算机内存严重来说包括 ROM，RAM，和寄存器
    • RAM 中的数据即使在断电的情况下也可以保留
    • ✔️
    • ✔️
    • ❌ 计算机内存严格来说包含 ROM RAM 和 缓存
    • ❌RAM 是随机存储，断电后数据会丢失。

  7. 关于计算机中数据的单位。以下描述正确的是

    • 一个字由诺干个字节组成
    • 一个字长为 8 位的存储单元可以存放 0~256 之间的任意一个无符号整数
    • 一个二进制位称为比特，通常用大写字母 B 表示
    • 微型计算机的字长是衡量计算性能的一个重要指标
    • ✔️
    • ❌ 一个字长为 8 的存储单元可以存放 0~255 之间的任意一个无符号整数
    • ❌ 位 称为比特 通常用小写字母 b 来表示 （bit）
    • ✔️

  8. 下列属于汉字输入编码的有

    • 区位码
    • 国际码
    • 五笔字型
    • 智能 ABC

  9. 为了提高 CPU 的处理速度，一般可采用的措施有

    • 增加 ALU 数量
    • 提高主频
    • 增加硬盘容量
    • 增加寄存器数目
    • ✔️
    • ✔️
    • ❌ CPU 不与硬盘直接交换信息，所以硬盘的容量大小与 CPU 的处理速度没有关系
    • ✔️

# 判断题

  1. 程序一定要读入到主存储器中才能运行。

  2. 以解决某一问题而设计的指令序列称为程序

  3. 中央处理器和主存储器构成计算机的主体，称为主机

  4. 计算机中最小的存储单位是二进制的一个位数

# 填空题

  1. 光计算机 是一种由光束进行数字运算，逻辑运算，信息存储和处理的新型计算机。

  2. 指令 是能够被计算机识别并执行的二进制编码，又称为机器指令。

  3. 外部存储器又称为 辅助存储 ，位于主机外部。

  4. 在进制计数中 数码 是指一组用来表示某种数制的符号

  5. 计算机硬件 是指计算机系统中由电子，机械，和光电元器件等组成的各种物理装置的总称。

  6. 时钟频率 表示 CPU 在单位时间内发出的脉冲数

  7. 计算机与其使用者的交互是通过 输入输出设备 来实现的

  8. 存储器 是计算机系统用来存储程序和数据的器件，它是一个记忆装置，也是计算机能够实现程序存储控制的基础。

  9. 用来记录存储信息的物质称为 存储介质

  10. 信息的符号化就是数据，所以数据是信息的 具体表现形式

  11. 机内码 是计算机内处理汉字信息时所用的汉字编码

  12. 用 MIPS 为单位来衡量计算机的性能，它用来描述计算机的 运算速度

  13. Cache 的中文译名是 高速缓冲存储器

  14. 指令 是一串二进制码组成，包括操作码和地址码两部分。

  15. 操作系统 是直接运行在裸机上的最基本的系统软件，而且其他软件必须在它的支持下才能运行。

  16. 软件 是指计算机运行所需的程序，数据和有关文档的总和。

  17. 运算器主要由 算数逻辑运算单元(或ALU) 和寄存器等组成。

  18. 将汇编语言源程序转换为等价的目标程序的过程称为 汇编

  19. 计算机中的算法具有有穷性，确定性， 可行性 ，零个或者多个输入，一个或者多个输出的性质。

  20. 存取器由成千上万个存储单元构成，每个存储单元都有唯一的编号，称为存储单元的 地址
