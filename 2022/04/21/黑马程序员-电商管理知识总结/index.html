# 使用阿里图标

  1. 把下载好的 font 文件放进 assets 目录下
  2. 在 main.js 中导入字体图标 import './assets/fonts/iconfont.css'
  3. 复制类名即可 <el-input prefix-icon="iconfont icon-3702mima"></el-input> iconfont 是固定写法

# vue2 中全局配置 axios

  1. 在 main.js 中导入 axios
    import axios from "axios";
  2. 设置请求的根路径
    axios.defaults.baseURL = 'http://127.0.0.1:8888/api/private/v1/'
  3. 把包挂载到 vue 的原型对象上
    Vue.prototype.$http = axios
  4. 通过 this 就可以访问原型上的 $http 发起 axios 请求

# 登录成功后保存 token

  1. 将 token 保存在 sessionStorage 中
    window.sessionStorage.setItem('token', res.data.token);
  2. 通过编程式导航跳转到主页
    this.$router.push('/home')
  3. 路由导航守卫控制访问权限

# 退出登录

  1. 清空 token
    window.sessionStorage.clear()
  2. 跳转到登录页面
    this.$router.push('/login')

# 通过 axios 请求拦截器添加 token

如果不通过拦截器添加 token 是这样写的～～果然学了拦截器后方便了很多呐
