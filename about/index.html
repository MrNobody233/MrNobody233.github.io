# 介绍

我叫 木子曰天 就一普通男的，没什么特长，想学的东西倒是很多，人也很懒，性格很怪，梦想是月薪 3000, 一个人勉强够花，不谈恋爱不结婚没关系，至少不想孤独终老，想遇见各种有趣的人，记得一个问题 人活着有什么意义 ，余华一本书《活着》中的回答加上自己的经历得出了自己的答案，那就是: 人活着本身没有意义,但遇到的每件事情赋予了人活着的意义 ，555EnjoySurprise~ 我太叼了，自我感觉良好。咳咳，喜欢两个词 Enjoy 和 Surprise, 享受我的生活和期待普通生活中能有些小惊喜，介绍就到这里结束啦 ovo, 掰掰～

我叫 木子曰天 ，公的，马上就要大三了已经大三了，还是一个什么都不会的废物。即使是废物也有自己的想法，在大一的时候一直在学习 JAVA，经历各种原因才确定目标，开始学习 Web 前端。梦想是用所学的东西来混口饭吃，有许多爱好，但无法保持热度，所以需要通过一生慢慢学习。由于我是一个古怪的普通人，同时也是一个懒鬼，所以介绍就到这里结束啦。
