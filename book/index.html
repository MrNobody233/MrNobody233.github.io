# 读者激励计划

咳咳，为了让自己能多读书，所以决定做一个重要决定！那就是每读完十本书就给自己一个奖励，到底是什么奖励呢，大概是能买喜欢的东西，像是好一点有线耳机，或者去吃一吃牛排之类的，不过十本书好像有点多哎，书的类型应该没有限制吧，主要是提高自己写作水平，让人家一看，哎哟不错哦，这个人有点文化之类的，所以不管是文学名著，还是科幻小说，只要是我感兴趣的都会算在里面，咳咳，肯定不会去看小黄书的啦...
