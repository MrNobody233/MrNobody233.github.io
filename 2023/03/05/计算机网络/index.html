# 单选题

  1. 下列关于 IPv6 的叙述错误的是

    • 地址空间更大
    • 首部格式灵活
    • 目前的主流操作系统以支持 IPv6
    • 不支持自动配置
    • ❌
    • ❌
    • ❌
    • ✔️ IPv4 和 IPv6 在设置主机的 IP 地址时都支持自动分配 IP 地址。

  2. 下列不属于网络协议组成要素的是

    • 语法
    • 语义
    • 时序
    • 进程
    • ❌用户数据与控制信息的结构或格式
    • ❌ 需要发出何种控制信息，以及完成的动作和作出的响应
    • ❌ 对事件实现顺序的详细说明
    • ✔️ 一个网络协议至少包括三要素：语法，语义，时序

  3. 计算机网络中，通信子网的主要任务是

    • 数据输入
    • 数据输出
    • 数据传输
    • 数据处理
    • ❌
    • ❌
    • ✔️ 通信子网提供计算机网络的通信功能，由网络节点和通信链路组成，主要任务是数据传输。
    • ❌

# 填空题

  1. 计算机网络是通过通信线路连接起来，在网络操作系统，网络管理软件及网络通信协议的管理和协调下，实现 数据通信 和 资源共享 的计算机系统

  2. 文件传输协议的英文简称是 FTP

  3. 电子邮件地址的格式为 <用户名>@<电子邮件服务器名>

  4. TCP 协议属于 TCP/IP 参考模型中的 传输层

  5. 为了解决信号衰减问题，实现对衰减的信号进行放大，应使用 中继器 设备
