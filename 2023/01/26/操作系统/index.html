# 单选题

  1. 利用 Windows 7 搜索功能查找文件时，正确说法是 。

    • 要求查找的文件必须是文本文件
    • 根据日期查找时必须输入文件的最后修改日期
    • 根据文件名查找时，至少需要输入文件名的一部分或通配符
    • 用户设置为隐藏的文件，只要符合查找条件，在任何情况下都将被找出

  2. Windows 7 的桌面是指 。

    • 放置计算机的工作台
    • 显示器上显示的整个屏幕区域
    • 放置显示器的工作台
    • 应用程序窗口
    • ❌
    • ✔️ 计算机启动完成后，显示器上显示的整个屏幕区域被称为桌面，桌面是用户与计算机交互的工作窗口。
    • ❌
    • ❌

  3. 在 Windows 7 资源管理器中，诺选择了 D 盘上的一个文件，并用鼠标左键将其拖拽到 E 盘中，其结果是将该文件从 D 盘 。

    • 移动到 E 盘
    • 复制到 E 盘
    • 删除
    • 剪切到剪切板
    • ❌
    • ✔️ 如果源文件或文件夹与目标文件夹不在同一个盘上，那么直接拖动可完成复制操作
    • ❌
    • ❌

  4. 以下不属于操作系统主要特性的是 。

    • 并发性
    • 共享性
    • 同步性
    • 虚拟性
    • ❌
    • ❌
    • ✔️ 操作系统具有并发性，共享性，异步性，虚拟性。
    • ❌

  5. 在 “资源管理器” 窗口中，选择 “工具” 菜单中 “文件夹选项” 命令后，不可以完成的操作是 。

    • 隐藏受保护的操作系统文件
    • 显示隐藏的文件，文件夹和驱动器
    • 隐藏已知文件类型的扩展名
    • 设置文件或文件夹的属性
    • ❌
    • ❌
    • ❌
    • ✔️ 设置文件或文件夹的属性是通过 “文件” 菜单完成的

  6. 以下有关 Windows7 操作系统的说法，错误的是 。

    • 自带防火墙，从而增加了安全性
    • 增加了对硬件的支持力度，支持所有硬件
    • 即插即用设备连接到计算机后，系统会自动检测到设备，并自动安装设备的驱动程序
    • 具有对用户账户的管理功能
    • ❌
    • ✔️ Windows7 支持通用硬件设备，但并不是支持所有的硬件设备。
    • ❌
    • ❌

  7. 利用 “控制面板” 的 “程序和功能”，可以 。

    • 删除 Windows 组件
    • 删除 Windows 硬件驱动程序
    • 删除 Word 2010 文档模板
    • 删除程序的快捷方式
    • ✔️ 利用 “控制面板” 的 “程序和功能”，可以打开或关闭 Windows 功能，查看已安装的更新，卸载程序，删除 Windows 组件等。
    • ❌
    • ❌
    • ❌

  8. 下列不是写字板可以保存的格式是 。

    • 文本文件
    • MIDI 格式文件
    • Unicode 文本文件
    • RTF 格式文件
    • ❌
    • ✔️ MIDI 是音频文件，写字板是 Windows 提供的文本编辑器。
    • ❌
    • ❌

  9. 在 Windows7 中被放入回收站的文件仍然占用 。

    • 硬盘空间
    • 内存空间
    • 软盘空间
    • 光盘空间
    • ✔️ 回收站是硬盘中的一块区域，可以存放硬盘上被删除的文件或文件夹，存放在回收站中的文件仍然占用硬盘空间。
    • ❌
    • ❌
    • ❌

  10. “在预览窗格中显示预览语柄” 在 对话框中进行设置。

    • 性能信息和工具
    • 文件夹选项
    • 索引选项
    • 管理工具

  11. 能够提供即时信息及轻松访问常用工具的桌面元素是 。

    • 桌面图标
    • 桌面小工具
    • 任务栏
    • 桌面背景
    • ❌
    • ✔️ 桌面小工具包括 “日期”，“时钟”，“天气”，“CPU 仪表盘” 等命令，可以提供即时信息，方便快捷。
    • ❌
    • ❌

  12. 在 Windows7 资源管理器中，要把 C 盘上的某个文件夹或文件移动到 D 盘上，用鼠标操作应该 。

    • 直接拖动
    • 双击
    • shift + 拖动
    • ctrl + 拖动

  13. Windows7 保存 “画图” 程序建立的文件时，默认扩展名为 。

    • PNG
    • BMP
    • GIf
    • JPEG

  14. 在大多数 Windows 应用中，打开 “帮助” 窗口最常用的快捷键是 。

    • F1
    • F2
    • F8
    • F9

  15. Windows7 对磁盘信息的管理和使用是以 为单位

    • 文件
    • 盘片
    • 字节
    • 命令
    • ✔️ 文件是存储在磁盘上的信息的集合，是计算机系统中数据组织的基本单位。
    • ❌
    • ❌
    • ❌

  16. 关于 Windows7 下属正确的是

    • 回收站与剪贴板一样，是内存中的一块区域
    • 只有对当前活动窗口才能进行移动，改变大小等操作
    • 一旦屏幕保护开始，原来在屏幕上的活动窗口就关闭了
    • 桌面上的图标不能按用户意愿重新排列
    • ❌ 回收站是硬盘里的一个特殊文件夹
    • ✔️
    • ❌
    • ❌

  17. 计算机操作系统通常具有的功能是

    • 进程管理，显示器管理，键盘管理，打印机管理和鼠标管理
    • 硬盘管理，软盘驱动器管理，CPU 的管理，显示器管理和键盘管理
    • 进程管理，存储管理，文件管理，设备管理和作业管理
    • 启动，打印，显示，文件存取和关机

  18. "控制面板" 无法

    • 改变屏幕颜色
    • 注销当前注册用户
    • 改变 CMOS 的设置
    • 调整鼠标速度

  19. 操作系统能有效地分配内存空间为多个程序服务，该功能属于

    • 进程管理
    • 存储管理
    • 文件管理
    • 设备管理

  20. 在 Windows7 中各应用程序间数据的交换和共享可以通过 实现

    • 资源管理器
    • 剪贴板
    • 任务栏
    • 快捷方式
    • ❌
    • ✔️ 剪贴板是 Windows 操作系统为了传递信息而在内存中开辟的临时存储区域。
    • ❌
    • ❌

  21. 下列关于 Windows7 附件自带的应用程序，叙述正确的是

    • “记事本” 可以处理文字，也可以处理图形
    • “画图” 是绘画工具，不能处理文字
    • “记事本” “写字本” 都不能处理图形
    • “写字板” 是文字处理软件，也可以进行图文处理
    • ❌记事本只能编写纯文本文档，无法处理图形
    • ❌ 画图是绘画工具，也可以处理文字
    • ❌
    • ✔️写字本可以进行图文处理

  22. 下列关于操作系统的主要特性的说法错误的是

    • 并发性是指两个或两个以上的运行程序在同一时间间隔段内同时执行
    • 共享性是指操作系统中的资源可被多个并发执行的进程所使用
    • 异步性，也称不确定性
    • 采用了并发技术的系统称为单任务系统
    • ❌
    • ❌
    • ❌
    • ✔️采用了并发技术的系统称为为多任务系统

  23. Windows7 中，关于文件和文件夹操作说法错误的是

    • 对文件和文件夹的重命名不改变文件或文件夹的内容
    • 被放入回收站的文件或文件夹可以恢复至原来位置
    • 移动文件或文件夹相当于对文件或文件夹进行 “复制”，“粘贴” 操作
    • 要创建一个文件夹，在资源管理器文件夹从窗口处右击鼠标，选择弹出窗口中的 “新建”,“文件夹” 即可
    • ❌
    • ❌
    • ✔️ 移动文件或文件夹相当于对文件或文件夹进行 “剪切” ，“粘贴” 操作
    • ❌

  24. 操作系统的功能负责管理系统的信息资源，可以实现存取控制

    • 进程管理
    • 存储管理
    • 设备管理
    • 文件管理

  25. 计算机能及时响应外部事件的请求，在规定的严格时间内完成对该事件的处理，并控制所有实时设备和实时人物协调一致地工作的操作系统是

    • 分时操作系统
    • 实时操作系统
    • 批处理操作系统
    • 分布式操作系统

  26. 在 Windows 7 中，shift + 空格 的作用是

    • 全角与半角状态之间的切换
    • 中英文的切换
    • 各种不同输入法的切换
    • 中英文标点之间的切换

  27. 操作系统将 CPU 的时间资源划分成极短的时间片，轮流分配给各终端用户，使终端用户单独分享 CPU 的时间片，有独占计算机的感觉，这种操作系统称为

    • 分时操作系统
    • 实时操作系统
    • 批处理操作系统
    • 分布式操作系统

  28. 关于 Windows7 中文件名的描述正确的是

    • 文件名只能为 8 个字符
    • 可长达 255 个字符
    • 文件名中不能有空格出现
    • 可长达 256 个字符

  29. Windows7 操作系统是

    • 多用户多任务操作系统
    • 多用户单任务操作系统
    • 单用户多任务操作系统
    • 单用户单任务操作系统

  30. Windows7 中，不可能在任务栏上出现的对象为

    • 对话框最小化后的图标
    • 正在执行的应用程序窗口图标
    • 已打开文档窗口的图标
    • 语言栏对应图标
    • ✔️ “对话框” 没有最小化按钮，只能在屏幕上移动，一般不能改变大小，也不能缩小成任务栏图标。
    • ❌
    • ❌
    • ❌

  31. 下列关于操作系统的描述中，错误的是

    • DOS 是单用户多任务操作系统
    • Windows XP 是单用户多任务操作系统
    • Windows 7 是多用户多任务操作系统
    • Unix 是多用户多任务操作系统
    • ✔️ DOS 是单用户单任务操作系统。
    • ❌
    • ❌
    • ❌

  32. 安装 Windows 7 操作系统时，系统磁盘分区必须为格式才能安装

    • FAT
    • FAT16
    • FAT32
    • NTFS
    • ❌
    • ❌
    • ❌
    • ✔️ 安装 Windows 7 系统的硬盘分区必须采用 NTFS 结构，要确保至少有 16GB 的可用空间，最好能提供 40GB 可用空间的分区供系统安装使用

  33. Windows 7 中，启动资源管理器，可以右键单击，从快捷菜单中选择打开 Windows 资源管理器 命令

    • 窗口的空白处
    • “开始” 按钮
    • 任务栏的空白处
    • 桌面看空白处

  34. 下列关于一些鼠标操作功能的叙述中，错误的是

    • 在文件夹图标上右键单击，可选择进入 “重命名” 操作
    • 双击控制菜单图标可以关闭窗口
    • 在 “开始” 按钮上右击，可选择打开资源管理器的操作
    • 在桌面窗口处右单击，可以选择对已打开窗口的排列操作
    • ❌
    • ❌
    • ❌
    • ✔️ 对已打开窗口的排列操作是在 “任务栏” 空白处右击，在弹出的快捷菜单内选择 “层叠窗口”，“堆叠显示窗口”，或 “并排先试试窗口”

  35. Windows 7 的库中由四个默认库组成，分别是文档库，图片库，音乐库和

    • 报表库
    • 数据库
    • 照片库
    • 视频库
    • ❌
    • ❌
    • ❌
    • ✔️ 默认情况下，Windows 7 已经设置了视频，图片，文档和音乐的库，用户还可以建立新类别的库。

  36. Windows7 中，打开 Windows 系统的 “开始” 菜单，可以使用快捷键

    • Alt + Tab
    • Ctrl + Tab
    • Ctrl + Esc
    • Ctrl + Shift

  37. 在 Windows 7 环境中，鼠标是重要的输入设备，而键盘

    • 几乎能完成所有鼠标的所有操作
    • 只能配合鼠标，在输入中起辅助作用
    • 根本不起作用
    • 只能在菜单操作中使用，不能再窗口操作中使

  38. 选择在 “桌面” 上是否显示语言栏的操作方法是

    • 控制面板中选择 “程序和功能”
    • 控制面板中选择 “区域和语言” 选项
    • 右击桌面空白处，选择属性命令
    • 右击任务栏空白处，选择属性命令

  39. 在 Windows 7 中，下列关于文档窗口的说法中不正确的是

    • 窗口中有文件夹和文件图标
    • 打开的文档显示在窗口中
    • 文档窗口用于编辑文档
    • 文档窗口是基于应用程序窗口而存在的
    • ✔️ 文档窗口中有标题栏，菜单栏，状态栏，滚动条等，没有文件夹和文件图标
    • ❌
    • ❌
    • ❌

  40. 在 Windows7 中，当一个在前台运行的应用程序窗口被最小化后，该应用程序将

    • 终止运行
    • 暂停运行
    • 继续在前台运行
    • 转入后台运行

  41. 在 Windows7 中，关于文件的 “共享” 正确的描述是

    • 所有文件夹都可以设置为共享
    • 含只读属性的文件夹不能设置为共享
    • 含隐藏属性的文件夹不能设置为共享
    • 所有文件夹都不能设置为共享，只有文件可以设置为共享。
    • ✔️ 文件夹都可以设置为 “共享” 属性，文件不可以设置为 “共享属性”，如果要共享文件，可以把文件放置到文件夹内共享。
    • ❌
    • ❌
    • ❌

  42. 在 Windows7 中，下列有关滚动条的说法正确的是

    • 用户可以通过控制面板设置，不允许滚动条出现
    • 滚动条是否出现，同窗口中的显示内容有关，只有特定的内容才出现
    • 滚动条在任何时候都会出现
    • 分为垂直滚动条和水平滚动条，只在需要的时候出现

  43. 在 Windows7 资源管理器窗口中，左窗口显示的内容是

    • 所有未打开的文件夹
    • 系统的树形文件夹结构
    • 打开的文件夹下的子文件夹及文件
    • 所有已打开的文件夹

  44. 在 Windows7 操作系统中，描述窗口的正确说法是

    • 对话框主要由边框，标题栏，菜单栏，工作区，状态栏，滚动条等组成
    • 用户只能移动文档窗口，不能改变文档窗口的大小
    • 窗口是 Windows7 应用程序的用户界面
    • 桌面也是 Windows7 的一种窗口
    • ❌ 对话框一般不包括菜单栏，状态栏等。
    • ❌
    • ✔️
    • ❌ 计算机启动完成后，显示器上显示的整个屏幕区域称为桌面，桌面不称为窗口。

  45. 在 Windows7 中，单击应用程序窗口左上角图标，可以

    • 关闭该窗口
    • 最小化该窗口
    • 最大化该窗口
    • 打开该窗口的控制菜单
    • ❌
    • ❌
    • ❌
    • ✔️ 标题栏左侧是应用程序图标，单击应用程序图标，会打开该窗口的控制菜单

  46. 剪贴板是

    • 磁盘上的一个文件
    • 一个文件夹
    • 内存中的一块区域
    • 磁盘上的一个特殊文件夹
    • ❌
    • ❌
    • ✔️ 剪贴板是 Windows 操作系统为了传递信息而存在内存中开辟的临时存储区域。
    • ❌

# 填空题

  1. Windows7 “资源管理器” 窗口左部的窗格是 导航窗格

  2. 在 Windows7 中，一旦选择了带省略号的菜单项，就会弹出 对话框

  3. 在 Windows7 中的 “资源管理器” 窗口左部，图标和名称前面带着空心三角的节点表明它含 下级子节点

  4. 在 Windows7 中的 “回收站” 窗口中，要想恢复选定的文件或文件夹，可以使用 “文件” 菜单中的 还原 命令

  5. 在 Windows7 中单击鼠标右键一般意味着打开 快捷 菜单

  6. 快捷方式就是一个扩展名为 .lnk 的文件

  7. 开始菜单 中存放着 Windows7 的绝大多数命令和安装到系统里面的所有程序，是操作系统的中央控制区域

  8. Windows7 中有三个不同类型的账户，即管理员账户， 来宾账户 和标准用户账户

  9. 在 Windows7 中，诺要将一个文件设置为 “只读”，则可以选中该文件，单击鼠标右键，然后弹出菜单中选 属性 菜单项

  10. 在 Windows7 中，运行 磁盘清理 ，可以释放磁盘上系统运行时产生的临时文件，Internet 缓存文件等占用的空间

  11. 在 Windows7 中，双击标题栏可以完成窗口的 最小化和还原 的切换

  12. Windoows7 对话框的控件中， 选项卡 多用于将一些比较复杂的对话框分为多页，实现页面的切换操作。
