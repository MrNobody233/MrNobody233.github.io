# 生日快乐

hello hello 此时此刻 2023 年 5 月 28 日 00:43:06，是我的生日，和以往一样是个普通的日子，今年倒是挺有意思，我的几位好朋友都记住了并且送上了祝福，今年第一个冠军是胖达，还挺惊喜，接着是暄，第三个是大个子，之前生日胖达和暄都不记得，一直是大个子第一，今年因为她在峡谷奋斗所以没拿上冠军，哈哈，排名其实并不重要，只是我每年这一天来为数不多的小小快乐罢了，很感谢他们的陪伴，希望未来能与他们分享更多快乐的事。记忆中我以前过生日时从不发朋友圈，好吧，平常也不发朋友圈，但是这次我剪了一个视频，我觉得特别好特别好，大概是从 b 站上看到一个意识流的视频，想着自己能不能剪一个出来玩玩，于是下载 PR，学习了十几分钟，翻遍了最近经常聊天和以前常聊天朋友的记录找合适的视频和照片，剪辑加上找素材花了快两小时左右出来了一个这样的视频，很满意，属于我自己的生活碎片，也许是 BGM 选的好，也许只是因为我喜欢回忆过去，翻记录时仿佛我重新经历了一遍快乐的时光，ButterFlyFish和我说了许多次: 如果总是回忆过去，可能是你对现状不满意。失去了然后去想那些曾经有的 ，不知道怎么说，我好像只是特别喜欢回忆而已，想重温一遍那时候快乐的时光，剪完视频后我才觉得，哇，原来我的人生过的其实还蛮精彩的，回忆过去对我来说好像就是饭后甜点，又或者是我害怕失去这段记忆，于是重复在脑海中播放加深印象，三个月前，我和大个子看《流浪地球 2》时，电影都还没开始，还在播放广告时，我就已经开始想象我未来回忆我与她看电影的记忆，然后我对自己说：好好享受这美好的时光吧，以后就很少见面了。就这样我开始享受这段时光，所以回忆过去也许并不全是我对现状不满意，只是我想在获得快乐的时候获得快乐，而回忆过去就让我感到快乐。

# 专升本

好吧 讲得有点离谱了，来写一下之前发生了什么事吧，好像有很长一段时间没有写博客了 (上面两个上锁的就不算惹～)，先从专升本开始吧，原本考完试就想写一篇的，但是有太多原因木有写，现在成绩也出来了，我考得好差，还以为能再高一些，结果也就那样，至少还努努力背了蛮久政治，发现自己只要下定决心的话还是会努力去做某一件事的，虽然做的不是很好，考试的前一晚，我为了不让自己太过紧张，看了一部电影，《疾速追杀》，针不戳，虽然我早就看过，一直在等他的第四部，看完后我就准备睡觉了，说是睡觉，其实也就是闭着眼睛躺了一晚上，我知道自己很难睡着，也准备好了考试时不会再像三年前单招考试紧张想吐的状态，所以我早上只吃了一口馒头，一口奶，不多不少，没有东西让我吐，就不会想吐，不想吐就不会紧张，去考场路上全是升本的学生 ，很心疼他们，又觉得好笑，因为他们都是短袖，南昌变天特别快，考试那天是刮风下雨的，我穿了外套和卫衣，有些人还穿着拖鞋出来，缩在角落瑟瑟发抖，我还遇到了我的室友，他也没有衣服穿，所以我大发慈悲把卫衣给了他，很有趣的是，这件卫衣就是我从他那里买的，我觉得很好玩，后面就不觉得了，少了那件卫衣我也有点小冷了。大多考生都是花 600 多元住的电竞酒店，当时情况有点特殊，有许多考试的，也有外地来看薛之谦演唱会的，所以酒店爆满，价格也翻了一倍，挺无语。考试大概就讲到这吧，既然已经考完了，那就准备好下一步该怎么去做了。

# 沉浸在自己的世界

考完试后我的堂弟就来奶奶家和我一起住了，这也是我不怎么写博客的原因，现在这篇也是在手机上写，怎么说呢，感觉周围有人我就不能好好写博客一样，就像考试时老师在你周围乱逛一样，哈哈，他开始来这里我还是蛮开心的，至少不是一个人在里面玩游戏了，有时候我也会带他玩派，过了几天后我就开始有点烦躁了，好吧，是很烦躁，特别烦躁，堂弟每次和同学打游戏时都会叫的特别大声，电脑亮度晚上也不会调小，对面丢一个闪光弹房间仿佛就像天亮，最早我会耐心的叫他不要吵了，爷爷奶奶在睡觉，也会去把他的亮度调小。到后面我就不管他了，不知道怎么说，他其他的都好，出去也会给我带吃的和奶茶，但只要戴上耳机就沉浸在自己的世界里，吵到我倒无所谓，我还可以熬夜，主要是怕吵到我爷爷奶奶，直到有几天我也熬夜和他一样中午起来，被我爷爷奶奶教育，后面我打开房门，他们才明白我起不来的原因，我奶奶昨天也跟他说叫他小点声，我也不记得我讲过多少次了，我这个人本来就神经衰弱，对声音特别敏感，昨天我玩派时他大声说话，我把音量调到 70，还是被刺激到了，瞬间爆炸，洗澡放松心情，在客厅呆了好一会，就这样吧，我也待不了几天了。

# 九江

五一时，我去了九江，特别特别开心，和我弟弟妹妹们一起玩，我奶奶也去了，因为我外婆 70 大寿，爷爷因为晕车加上我堂弟的外婆也过寿，所以两边各去一个，五一发生了什么，让我回忆一下，噢，是我外婆大寿，我坐小孩那桌，很无奈，看着小孩们玩手机，还没上几个菜就动筷子，感觉很没教养，但他们都是客人，他们走后，我们打扫卫生，我大表妹坐在哪玩手机，服了，但是看她那么久没玩我就没管她，到晚上，她非要跟我玩捉迷藏，我跟她约了赌注，我输了就给她玩手机，我赢了就打她屁股十下，我跟她玩了三次，她三次都没抓到我，我愿给自己一个躲猫猫捉迷藏高手的称号，所以我打了她十下屁股，不算轻，就当是她下午不跟我们打扫卫生的惩罚。

接着是看小姨提亲，舅舅舅妈还有我三个人去，我不知道为什么要叫我，后面问了一下是说要人数凑双数，不能凑单数这个习俗，所以我们这边是十个人（加上我姨婆那边的人），准备去姨婆家时，舅舅舅妈各开了一辆车，我布吉岛坐哪辆，我阿姨把我推到舅舅那辆车，估计是想让我和舅舅多聊聊天，实际上在舅舅车上我也没说什么话，一路上在内心感慨舅舅的车坐的好舒服，空间好大，很商务，唯一的对话就是他告诉我车门边上有水可以喝。到了姨婆家，我看见小姨穿的特别漂亮，回忆起小时候他们的房子还没拆时，我和我表弟和她一起玩小猫钓鱼，一直以来对她的印象就是漂亮温柔。时光飞逝她现在已经被提亲了，我很紧张，仿佛提亲的是我一样，我阿姨他们也调侃我，说舅舅让我过去是学习经验的。然后就是去饭店吃饭，妈呀，大早上就吃那么好的，是我完全不敢想的，去的饭店是我姨婆工作的地方，她待人很好，给饭店每一个工作人员准备了喜糖。然后就是我当工具人的时候了，舅舅叫我来帮姨婆给桌上每个人发喜糖，到我座位时，我还小声问我有没有份，都把人家给逗笑了，那肯定是有的。然后是倒饮料，没错又是我，我舅舅说的比较大声，实际上在提示新郎，让他去倒，他开始还没反应过来，还是被我小姨提醒了。我舅妈说他反应迟钝，他给我倒饮料时，我差点绷不住，小姨向他介绍过我了，名字从他嘴里说出来感觉有点搞笑，给我倒饮料时我一直在想怎么感谢他，然后就是脱口而出的 “谢谢小姨夫”，现在回想起来真的挺搞笑，吃完饭后是要去男方家，我坐的是舅妈的车，在路上听舅妈和姨婆聊天，听见她说那个男孩子还没结婚就喊她妈，脸皮真厚，我直接绷不住，笑出声来。

早在过年时，我就见到了这个 “小姨夫”，开始我还以为是我小姨的什么亲戚弟弟，吃完饭回家了才知道我擦，这是小姨对象。我姨婆很反对这门亲事，在提亲前几个月还是这个态度，后面想了想，也听了我外婆的话，大概就是如果不让小姨和他在一起，未来小姨和别人结婚过得不好，会怪罪我姨婆的。到这里我就没话讲了，这也是在去男方路上听我姨婆说的，体会到了做父母都无奈，我舅妈也说结婚是两个家族的事，不是两个人的事。我沉默了，好吧，我本来就没说话，我是内心沉默了 ⁽⁽ (๑˃̶͈̀ ᗨ ˂̶͈́) ⁾⁾，到后面姨婆说她再也不讲了，既然已经提亲了，她该讲的已经和我小姨讲了，以后就不提了。我也怎么没想到这么温柔的小姨也有叛逆的一面 哈哈。

经历了山路十八弯，终于到了男方家，一股烟花味开始扑鼻而来，好嘛，男方家这边的习俗是来的每一家客人都会带一个烟花过来，所以门口堆满了烟花，没吃多久早饭又开始吃中饭了，这次我坐的是长辈那桌，也就是男方父母那桌，边上是我舅舅舅妈，我表面毫无波澜，内心笑嘻了，妈的提亲真好玩，人家爷爷来给我敬酒，每一个长辈都轮着给我敬酒，因为我的辈分很小，所以我需要站起来跟他们碰酒，这样一想桌酒文化我还是明白个大概的，也许过不了多久我就变成自己讨厌的模样，主动给长辈们敬酒吧。

然后就是饭后闲谈了，趁着 “小姨夫” 给我们拿饮料，我问小姨之前没来过这里吗，她说这是第一次来，妈呀给我震惊了，我小姨真这么傻的嘛，我姨婆都提前来考察过了，男方门口的枣子树都摘过，我小姨居然还是第一次来，这有点让我怀疑小姨是不是恋爱脑，我的内心发生了一点变化，从最最最开始，过年那段时间，姨婆反对，说男方太丑了之类的，我在内心说不觉得颜值是个问题，对待我小姨好就行了。直到现在我的想法发生了一点变化，或许是我小姨想的太简单美好了，或许是我姨婆了解的还不够全面，明年他们就该结婚了，我对这件事情后续发展表示特别关心。回家路上听小姨的哥哥也就是我小舅舅和他老婆说，“小姨夫” 和小姨几乎很少出门约会，出门去玩还是 AA，打车参加我外婆大寿时，我小舅舅转车费给他，他居然还收了。有点意思，恋爱开始 AA 倒是很正常，但是都快结婚的人了，这样是不是有点不太对了，难道这就是新型恋爱的方式吗，路上舅妈也说男方家太偏僻了，连个公交都没有，如果小姨想要回来是一件非常困难的事。我布吉岛怎么说了，只想知道后面会怎么发展。

送完姨婆他们回家后，舅舅把男方那边给的红包给了我，好耶.600 元。舅舅当媒人给的 1000 元。小姨是 10001 万里挑一，第一次见那么大的红包，然后我们准备回去了，舅舅要买什么东西，让我跟舅妈先回去，这也是我和舅妈说话最多的一年，我很佩服我的舅妈，提亲前几天坐舅妈车去饭店吃饭时，她叫我开她的车，我觉得她在逗我玩，我肯定得拒绝，回去跟我爸妈聊天时，提了一句，他们说那肯定不能开呀之类的，我也是这么认为的，并不是我舅妈的车很贵，我只是担心我大表妹也在车上，我害怕出事，后面出门她也提了几次 说她的车很好开，还把手离开方向盘，说她的方向盘很稳，不会乱动，舅舅的就会动。这次回家路上，和她闲聊，她也不提开车的事了，直接把车停在路边，让我来开，我惊呆了，我一直以为她是逗小孩玩，从没想过她真的把车给我开，而我也确实很想开，这是一个很好的机会，路上几乎没车，就我和舅妈，因为我学的是手动挡，她简单的讲解了一下自动挡怎么开，我坐在驾驶室重复着她说的话，证明我知道怎么去开了，然后就是起步，我不敢开的很快，我舅妈叫我加速，然后我开到快 60 码，我测，太刺激了，掉头转弯什么的也试了一遍，路上还遇到逆行的大货车，我减速打右转向灯变道，顺便还按了两下喇叭，不得不说我学习开车的能力还是很强的，除了我空间感有点差，不知道车是不是在道路中心，到了回家的那条路，就下车交给舅妈开了，然后我的腿开始发抖了，真刺激。

# 现在和未来

之前的事大概就这些了，该讲一讲现在了，最近买了两个游戏《幽灵行者》和《泰坦陨落 2》怎么说呢，没花多少时间就通关了，也许是有 APEX 基础。泰坦陨落模式还是最困难的，开始死的次数还挺多，习惯以后就差不多了，幽灵行者时间就长一点点，原因是有些关卡玩的头晕想吐，妈的最讨厌旋转的地图惹。玩完以后又感觉没事做了，apex 也上了大师，感觉没什么意思，偶尔也会和朋友打打靶场，不知道他怎么想的，好像有受虐倾向，我还害怕他自闭，没认真跟他玩，但还是随便打打就打过。发现这样不会减少他对这个游戏的热爱，也认真跟他打过，谦虚总结一下就是他杀我一次我能杀他十次吧... 太过于无聊了，也下过王者荣耀，但因为一些原因快装好了又删除了.

六月份我就该去合肥了，明明挺开心的，马上迎来新的生活，又感觉有许多遗憾，好像有很多事没做完，上了这么久的学就这样结束了？就这样进入社会了吗，是有什么东西没有给，有什么秘密花园没给人家装，在走之前把事情就解决好，还是留着未来去完成。最近很少和谁去聊天，做什么都提不上精神，估计这就是在家呆久了，越呆越郁闷，出门散会步后感觉好了许多，想起在九江聊天时舅妈说一定要出去工作，要有自己的圈子，整天呆在家里会呆出病的，虽然这本来是对那个不想出去工作的小舅妈说的话，但是我也听进去了

前段时间还和胖达去看了《银河护卫队 3》，以前经常和他一起看电影，《流浪地球 1》就是和他一起看的，怎么说呢他作为我的好哥们即使不怎么在微信上聊天，但是线下却是见面最多的，一年至少得两三次吧，虽然对于其他人不算多。而且说去干什么那就只是去干什么，就像这次去看电影，看完电影后我们就各回各家了，我感觉很不错，也许这就是男人的友谊吧。

突然就不想写了，最近几天都是重复的做几件事，今天就让着自己把，今天我是祝福自己的第四名，还挺多人，我想该给自己买几件好看的衣服了，我的裤子后面还破了个洞，衣服穿的还是高中买的，就这样叭，买点好看得衣服让自己开心一下，生日快乐，掰掰～
