# 单选题

  1. 1KB 的存储内存可以存放个汉字机内码 。

    • 1024
    • 1024 x 512
    • 512
    • 251
    • ❌
    • ❌
    • ✔️ 一个汉字机内码占用两个字节 (2B)，1024/2 = 512 个字节
    • ❌

  2. 在 Windows7 回收站中，可以恢复

    • 从硬盘中删除的文件或文件夹
    • 从 U 盘中删除的文件或文件夹
    • 剪切掉的文档
    • 从光盘中删除的文件或文件夹
    • ✔️
    • ❌ 从 光盘 ， U盘 中删除的文件或文件夹不进入回收站，而是被彻底删除
    • ❌ 剪切掉的文档被暂时存放在剪贴板中，不会进入回收站
    • ❌

  3. 在 Word 2010 中关于页眉和页脚的设置，下列叙述错误的是

    • 允许为文档的第一页设置不同的页眉和页脚
    • 允许为文档的每个节设置不同的页眉和页脚
    • 允许为偶数页和奇数页设置不同的页眉和页脚
    • 允许为页眉，页脚和文档内容同时编辑
    • ❌
    • ❌
    • ❌
    • ✔️页眉和页脚不能与文档内容同时编辑，但可以一起打印

  4. 在 Excel 中，公式 =SUM("3",2,TRUE) 的计算结果为

    • 2
    • 3
    • 6
    • #VALUE!
    • ❌
    • ❌
    • ✔️ 诺公式中的参数为逻辑值或能被转换为数字的文本，则将被计算在内。逻辑值 TRUE 对应数值 1，FALSE 对应数值 0
    • ❌

  5. 要使用电话线上网，计算机系统中必须要有

    • 声卡
    • 网卡
    • 电话机
    • Modem
    • ❌
    • ❌
    • ❌
    • ✔️ Modem (调制解调器) 是电话拨号上网必备的设备。

  6. 局域网不包括的特点是

    • 地理范围有限
    • 传输倍率快
    • 误码率低
    • 传输延迟大
    • ❌
    • ❌
    • ❌
    • ✔️ 局域网覆盖范围有限，一般从几百米到几公里，传输倍率高，可靠性好，传输延迟小，误码率低，适用各种传输介质，建设成本低。

  7. 诺在某一个工作表的某一个单元格中出现错误 #NAME? , 可能的原因是

    • 用了出错误的参数或运算对象类型，或者公式自动更正功能不能更正公式
    • 单元格所含的数字，日期或者时间比单元格宽
    • 公式中使用了 Excel 中未定义的名称
    • 公式被零除
    • ❌
    • ❌
    • ❌
    • ✔️ #NULL! 表示交集为空， REF! 表示引用无效单元格， VALUE！ 表示不正确的参数或运算符。

  8. 关于 WAV 文件，不正确的描述是

    • WAV 是波形文件
    • WAV 是 Microsoft 公司制定的音频文件格式
    • WAV 文件进行剪辑后会失真
    • WAV 文件是一种非压缩的音频文件
    • ❌
    • ❌
    • ✔️ 由于没有采用压缩算法，因此无论进行多少次修改和剪辑都不会产生失真。
    • ❌

  9. 在计算机系统内部处理和存储汉字字符码采用的编号是

    • BCD 码
    • 国际码
    • ASCII 编码
    • 机内码

  10. 下列描述中正确的是

    • CPU 可以直接执行外存储器中的程序
    • RAM 是外设，不能与 CPU 交换信息
    • 外存储器中的程序只有调入内存后才能运行
    • Flash 存储设备是一种易丢性半导体存储器
    • ❌ CPU 不能直接执行外存储器中的程序，必须将外存储器中的程序调入内存才能被 CPU 执行
    • ❌ RAM 是内存储器，可以直接与 CPU 交换信息。
    • ✔️
    • ❌Flash 存储设备简称闪存，是一种非易失性半导体存储器。

  11. 在 Word2010 中，对于一段两端对齐的文字，只选定其中的几个字符，用鼠标单击 “居中” 按钮，则

    • 整个段落均变成居中格式
    • 只有被选定的文字变成居中格式
    • 整个文档变成居中格式
    • 操作无效，格式不变
    • ✔️ “居中” 是 段落 的对齐方式，针对的是选定字符所在的 段落 。

    • ❌

# 多选题

  1. 关于 ASCII 码字符集中的字符，下面叙述错误的是

    • ASCII 码字符集共有 128 个不同字符
    • 每个字符都是可以打印的
    • ASCII 码最高位可用于奇偶效验
    • ASCII 码字符集中，大小写英文字母的编码相同
    • ❌
    • ✔️ ASCII 中有 33 个字符是控制字符，它们在传输，打印，或显示时起控制作用，是不能打印或显示出来的
    • ❌
    • ✔️ ASCII 中大写英文字母的编码 + 32 = 小写英文字母的编码，两者并不相同。

  2. 安装 Windows7 时 (32 位) 推荐的最低硬件配置，包括

    • 处理器：1 GHz
    • 系统内存：1 GB 及以上
    • 硬盘分区：40 GB 及以上
    • 显卡：需支持 DirectX 9 及以上
    • ✔️
    • ✔️
    • ❌ 16GB 可用的硬盘空间 (32 位) 或 20GB 可用的硬盘空间 (64 位)
    • ✔️

  3. 关于汉字外码，下列说法不正确的是

    • 汉字外码是用于将汉字输入到机器内所使用的的编码
    • 一个好的外码应具有编码短，重码少，好学好记等特点
    • 汉字外码一定是等长编码
    • 汉字外码有音码，形码，字模点阵码
    • ❌
    • ❌
    • ✔️ 汉字输入码编码方案有多种，汉字外码不一定是等长编码。
    • ✔️ 字模点阵码属于汉字字形码，不属于汉字外码。

  4. 下列属于区块链基础技术的是

    • 哈希运算
    • 数字签名
    • 共识算法
    • 智能合约

    • 129.10.20.5

    • ❌
    • ✔️ B 类 IP 地址的首字节范围为 128~191
    • ✔️
    • ❌

  6. 在演示文稿的幻灯片浏览视图中，可以进行的操作有

    • 新建幻灯片
    • 修改幻灯片中的文字
    • 删除幻灯片
    • 选择幻灯片切换效果
    • ✔️ 在幻灯片浏览视图中，屏幕上可以同时看到演示文稿的多张幻灯片的缩略图，用户可以新建，复制，删除幻灯片，调整幻灯片的顺序，设置幻灯片的背景和切换效果，但不能对单张幻灯片的内容进行编辑，修改。

    • ✔️

# 填空题

  1. 在媒体的定义中，光盘应属于 存储媒体

  2. 一个汉字的国际码为 56 50H，则该汉字的机内码用十六进制数表示应是 D6 D0

  3. Word 的 艺术字 功能可以创建带阴影的，扭曲的，旋转的，和拉伸的文字，可以按预定义的格式创建文字。

  4. 在 Excel 2010 单元格中输入公式：="10" & "20", 按 Enter 键后，单元格中的显示结果为 1020

  5. 作为电子邮件服务器要用到的两个协议分别是 SMTP 协议和 POP3 协议

  6. 导弹发射系统是指用于支撑导弹并决定其射向，发展发射准备和实施发射的专用装置，对信息处理的过程控制有着严格的时间要求，这种操作系统属于 实时 操作系统

  7. 打印 Word 文档之前最好能进行 打印预览 以确保取得满意的打印效果

  8. 信息安全 是指信息网络的硬件，软件及其系统中的数据受到保护，不受偶然的或者恶意的原因而造成的破坏，更改，泄露，系统连续可靠正常地运行，信息服务不中断。

  9. 如果要分别统计男女生的总分平均值，应使用 分类汇总 功能。

  10. 一台计算机有许多指令，作用也各不相同。所有指令的集合称为计算机 指令系统

  11. 数据有效性 是 Excel 的一种功能，用于定义可以在单元格中输入或应该在单元格中输入哪些数据，可以避免一些输入错误。

  12. 按照国际电话电报咨询委员会对媒体的定义，动画和投影仪分别属于 感觉 媒体和 显示 媒体。

  13. WWW 采用 客户机 服务器和工作模式
