# 过年

因为疫情很久都没有回外婆家了，所以今年过年回外婆家过，都挺有意思，外婆家可以放烟花，奶奶那边家人很多，但是奶奶那边以后不会聚在一起过年了，因为家族人太多了，没有地方可以办酒，还记得以前过年每个房间挤满了人，但近几年小孩子聚不到一起去了，感觉很陌生，爷爷叔叔那辈倒是和以前一样，打牌的打牌，打麻将的打麻将，小孩子就吃完了饭在那里玩儿一会手机然后就自己回家，也许大家都长大了吧，面对尴尬的亲戚问候还不如去找自己的朋友拜年。

今天是 2 月 1 号，我在外婆家住了 15 天，时间过得很快，半个月就过去了，我过得很开心，很悠闲，也很享受和家人待在一起的时光，在外婆家睡眠都挺规律的，指的是晚睡早起，并且会吃早餐，和奶奶不同的是外婆会强制要求我吃，而奶奶会惯着我，因此我来外婆家后拉的屎也比较多，晚上睡觉也会放屁，我弟弟很有先见之明，在我来之前他就铺好了两床被子，不然会污染他的被窝，但无法避免的会污染房间内的空气，有时候晚上我还会掀开他的被子把屁放他被子里，哇，太贱了，关键是还贼臭，味道怎么说呢，好像就是把粑粑放在微波炉里转了三分钟飘出来的味道。哈哈哈哈，比喻鬼才就是我了。

# 可爱的表妹们

在外婆家还可以陪小表妹玩，她今年四岁，小孩子简直太可爱了，她拉着我的手，给我玩小孩子的玩具，然后给我扎头发，模仿大人的语气叫我不要动，但小孩子不是一直都那么可爱，有时会她也会很哭闹，比如她打翻了瓜子死不承认，舅舅舅妈还有外婆三个人一起教育她，把裤子拔下来然后拿皮带抽，她也不承认错误，还哭得贼大声音，好在半小时后她还是捡起瓜子并承认错误，有时候外婆不让她看电视她也会哭，哭得特别伤心后外婆也就心软了，会叫我放给她看，但并不是每次都会心软，所以我就出马了，只要我陪她玩儿她就不哭了，还会笑的特别开心，如果她哭了我就会跑的远远的，我喜欢小孩子但不喜欢吵闹的小孩子，我在我奶奶家住的时候听见下面的臭小孩尖叫，我就幻想着我拿着他爸爸的皮带狠狠地抽他的屁股。所以和小表妹相处的这些天，觉得她被教育的很好，但却让我不想生小孩子了，感觉以后没那么多时间，也没有耐心。

然后就是大表妹了，她现在已经十一岁了，梦想是想当兵，喜欢玩和平精英，刷的抖音全是和平精英的内容，老是叫我玩这个游戏，她说只要看着我玩就行了。然后就演变成我和她组队玩和平精英，并被她爸爸一起揪耳朵... 除了游戏我还会和她玩其他的游戏，比如捉迷藏，赌注就是抓到我了就给她玩儿和平精英。还有晚上放烟花，她也不敢放，只会捂着耳朵躲在门缝后面看，有一天小表妹扎头发和大表妹放烟花起冲突了，她们都想要我陪她玩儿，我坐在沙发上让小表妹扎头发，大表妹让我和她去放烟花，小表妹不让我走，大表妹说她失宠了，哎妈呀，给我整无语了，舅舅说陪她放，她也只想让我陪她放，嘿嘿，看来我还挺受小孩子喜欢的，我大表妹说我长的很好看，什么时候带个女朋友回来，又给我干沉默了，但我可不是好惹的，直接开始反击，说等她当完兵回来我已经结完婚了，孩子都和你现在这么高了，不知道她以后对当兵这件事会不会产生一点点犹豫，嘿嘿。她们姐妹俩要是吵起来可不会手下留情，大表妹会打的特别用力，但是小表妹会和没事人一样，难道是打多了？还是穿的太厚了。。

# 又是贴对联

除夕那天好像是在贴对联，哎，又是贴对联，简直就是折磨，这次是外婆家，大门口贴大对联，站在梯子上面拿大胶带贴，真不好贴，外婆家的那个墙是粉末状的，贴上去后会把墙粉带下来，很难粘住，但还是硬着头皮贴完了，当我们以为结束的时候其实才刚开始，还有上面的也要贴，里面所有的房间也要贴，好在房间内的对联已经贴的差不多了，不然外婆家三层楼里面十几个房间真的会崩溃的，贴对联之前还挂了灯笼，贴福字什么的，装饰完后年味又升了一个档次，虽然外面的对联已经被风刮走了，刚才还因为这个被舅舅教育了。不过这一切都是值得的，因为我可以用贴对联来水一篇作业，叫什么劳动教育，也就是写自己在家帮了什么忙之类的，真是闲得蛋疼没事找事。。

# 习俗和拜年

外婆家的习俗和奶奶家不一样，外婆这边春节早上要先请祖宗吃饭，于是我们春节早上就开始给太外婆太爷爷上香烧纸什么的，还爬山去给外公烧，爬山才知道我身体有多差，没爬一会儿就累的要命，当我终于到顶后才发现，，，这个山可真是坟山啊，一眼望去全是坟。晚上吃完饭就坐在那里看春晚，今年春晚也是一坨答辩呢，尬的我抠脚，舅舅叫我们别玩手机，看会儿春晚，所以我硬着头皮看了个大概，那个一分钟脱口秀，，麻的那个人站在那里不动都比他讲话好笑，整个春晚唯一有意思的地方好像就是那个微电影，所以到后面我舅舅也放弃了，大家一起看手机。

去姨婆家拜年的时候会路过龙城，那里离废宅充电器家很近，想着有没有机会去薅一下他家的羊毛，所以给他发了个地址，没想到他居然起来了，还以为他这个阴间人会下午才回我消息，谁知道他不是起来而是没睡，不愧是阴间人，所以和我爸妈提了一句，他们说这边的习俗下午不能去别人家拜年，除非家里有人过世了，巴拉巴拉的，废宅充电器问了他妈，说他们家是河南那边过来的没有这个习俗，只有初四不能来拜年。所以我还是去找他玩了，并听爸妈的话买了一箱王老吉一箱奶，花了我 90 软妹币，等待一会儿终于看见了他，长发油头，真不错，我有点庆幸我回外婆家之前剪了头，不然他爸妈会以为现在小孩子都喜欢留长发。他带我去看看楼下的牛奶猫，太可爱了，但我不敢多摸它，因为我大表妹去她外婆家时被猫抓了，下午要去打狂犬育苗。去他家之前还要爬楼，真是折磨啊，还是在最顶楼，他说他家厕所很大，我想厕所能有多大，因为尿急还抱这个好奇心我克服了爬楼的困难，问候完他爹妈奶后就去上厕所，真没骗我啊，厕所 + 洗漱台能有一个房间那么大，我靠真是设计鬼才啊，然后他带我去他的电竞房玩空洞骑士，期间他爸爸把瓜子花生和茶端进来，我和充电器就坐在暖脚器上盖着被子，用手柄玩。这可比拜年有意思多了，我还拍充电器给其他同学看，笑嘻了， ButterFlyFish还以为是我表妹，果果超甜说勉强能冲，看来充电器真该剪剪头了。手柄玩游戏确实很舒服，但是他的电竞房没有床我不是很认可，并且对他爸妈不管他的阴间生活表示怀疑，后面我爸妈做完客后就带我走了，因为说要拍一下合照。还是在朋友家玩有意思，前段时间连续阴天都快给我整抑郁症，甚至在上香时候其他人都是保佑学习进步，而我在心里默念保佑自己身体健康和心理健康。

# 摘野菜

有时候会去长江附近摘野葱，不过都是我妈摘得，我们看着她摘，在那条路上我突然听见熟悉的声音，玛德实在是太熟悉了，我立马就想起来了，是荒野大镖客的火鸡叫，卧槽，居然是真真的，有人在养火鸡，荒野大镖客诚不欺我，声音一模一样，我也能在游戏里学到知识啊，呜呜。然后就是我的大表妹，明明才十一岁，就已经知道了许多梗，不知道从哪里学到的，说着小飞棍来咯，这句话，我把她录了下来，等她结婚的时候就给她放出来，现在的小孩可真好啊，黑历史都是高清的。不过我很开心她还能和我玩到一块去，我还以为她大了以后就不会和我们玩到一块去，不过也是我舅舅舅妈的教育方式好，不会给小孩子玩手机，这也导致了她会缠着我让我给她玩。她还给我做了许多微信表情包，虽然我有点邋遢，但我还是会留下来嘻嘻。

# 明天的事

在外婆家住了这么多天，多少还是有些不习惯的，也许是我的主机不在这里，也许是两个人睡在一起不习惯，反正舒服还是很舒服，我经常坐在外面晒太阳，很温暖，各个方面的温暖，可惜最近几天又要变天了，希望我的心情不会变，原本今晚想一个人去看看 《流浪地球 2》的，但是大家一起在奶奶家吃的饭，不过正好我高中老师起死回生了，可以去拿我心心念念的牛肉干和毕业证还有毕业照了，所以我会和大个子一起去看看老师顺便看看电影和滑会儿滑板，这也许是我寒假结束前最后一些乐子了，马上就要开学了，希望学校没那么多破事，就这样，其实还有很多想写的，不过我答应我奶奶早上八点半起来吃早饭了，就这样，掰掰。
