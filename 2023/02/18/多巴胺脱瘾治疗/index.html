# 身份证

都出门一段时间了，才发现现在是正月，看来得过两天才能剪头了，烦内，这次的头发长的特别丑，原因是很早之前我身份证快过期了，当时我头发很长，又没洗头，于是和我妈妈去理发店，但是如果剪头的话我的头发又会很难看，妈的，这可是身份证啊，要用十年的身份证啊！为了我帅气的形象，我犹豫不决，于是我妈就在理发店帮我洗头，尬死我了，虽然理发店的老板是我姨夫的徒弟，但还是好尴尬。然后我妈妈给我吹头发，不知道形容，好像就是把我吹成了路边上长相一般的头发比较长的男的，呜呜呜。在去派出所的路上我疯狂的把自己的头发往上撩，颜值上去了那么一丢丢，然后在派出所自助办理身份证机器里拍照，好嘛，它判断我头发太长遮住耳朵和眉毛之类的，拍了好几次都不行，真是麻麻石蜡。所以那天还是没办成身份证，过了两天后我就去理发了，不知道怎么回事，好像是我妈妈在边上，他给我剪的头发是那种很乖很小孩子的头发，更加无语了，所以我又得拖延一点时间让我头发慢慢成长了。接着就是去我外婆家呆了半个月，回来后过了一段时间，终于找机会去照相馆拍身份证照了，第二天早上去拿照片，麻的，这是谁啊，死鱼眼，又板着脸，我真的好不甘心啊，于是我去网上搜哪里能办身份证数码照片回执，找到后我自己上传了自认为帅气的照片，然后网站就开始给我返回各种错误，什么距离太远，颜色异常，上传三四次后网站就开始繁忙了，尼玛真的搞我心态啊，我又开始找其他办法，功夫不负有心人，在赣服通里有居民身份证换领，简直太方便了，一共花了我 35 元，虽然照片是之前的照片，不过还好上个身份证的照片还算不错。身份证拿到手的那一刻，终于放松了起来。

# 情人节和第一次收到花

玩游戏，玩游戏，玩游戏，为什么强度这么高啊，大神难道都没有对象的吗，果然女人只会影响拔枪的速度。那天我在 BB 空间看见了她第一次收到花，好像是谈了恋爱，我有点小失落又有点小开心，之前我过生日的时候她是人生中第一个送我花花的女孩子，我很开心，我就想着我一定要找机会送她一次花花，甚至送什么花都想好了，我从来没送过同龄女孩子花花，我觉得麻烦，过不了几天就枯萎了，那还不如不送。但是我收到花的时那份开心才让我认识到送花的意义。我记得小学教师节时送了许多老师花花，还是很漂亮很贵的那种，老师们那发自内心的笑容让我一直记到现在，所以我还是会第一次送花给她，不为别的，就因为她是第一个送我花的女孩子。就算她不是第一次收到花，那也是我真正意义上第一次送出去的花。然后那天晚上就一家人去吃火锅，突然就好想找个人聊天啊，线上打字太累了。于是就找了个买衣服和鞋子的理由叫大个子第二天陪我出来逛街，我给她半个小时时间回复去或者不去，因为半个小时正好是从火锅店走到奶奶家的时间。她说她去，也确认了她是闲得无聊没事干才去的。接着回去打游戏到凌晨四点才睡，呜呜因为游戏更新了，我最喜欢的人物被削弱了。

# 饼干

啊～熟悉的地方，好像是叫奥特莱斯，高中时常和各种人在这里逛街，和大个子好像还是第一次，她给我带了亲手做的饼干，看着非常人畜无害，当我咬下去细细品尝时...？好像是在嚼一个没有味道的土，然后土里面有几颗蔓越莓。看来我的形容又更上一层楼了。她说她忘了加糖，我只能说我不懂怎么做饼干，但还是想说 6 。直男就是我，我就是直男。石头剪刀布她输了，所以她请我喝 CoCo。于是我们就坐在里面聊天，聊了什么，我忘了，各种各样的事，非常有意思，她叫我专升本到她学校，呜呜，别提了，只要能达到分数线，什么学校我都可以接受。还好她还年轻，没那么早挂，希望她多遇到点有意思的事，然后继续讲给我听。然后就是去逛街了，其实我确实想买鞋子的，只能说我眼光太高了，去了好多家我都没有看上，耐克和阿迪达斯的鞋我又买不起，感觉大个子快被我折磨死了，哈哈哈哈，算了，还是回去线上购物吧。然后就去吃海底捞，就是很想吃，再顺便看看我老师的剧本杀店，妈的居然关门了，不然我要砸他场子！海底捞的小姐姐带我们进去，我要尬死她，问了一句可以做美甲麻，姐姐好像被我沉默释放不出技能了，说了一句要先预约。真好玩，口罩真是个好东西，下次继续找个人尬。我点了蛮多东西的，也许是我熬夜玩游戏，导致我战斗力下降了，所以我状态不是很好，虽然我特别想吃，但是我又困又累。还好大个子战斗力还不错，出乎我的意料，虽然没有 X 在长沙吃火锅时牛逼，但还是比我强得多。对了，我还带了滑板给她滑，本想看她出丑的，结果她居然会滑，只是不会转弯而已。死去的回忆突然攻击我，想起高中带滑板来学校玩，有个女孩子玩我滑板摔跤了，我没有管她，而是站在边上看着她还问了句还要玩吗，她说还要玩。我把这个记忆告诉了大个子，大个子说她就在现场，啊？啊？啊？然后我就说了一句原来那个时候你存在感这么低，wow，爆杀了，原来我还是和以前一样那么令人无语。然后干了啥，哦哦哦，每日一张，和上次不一样的是她拍的更好看了，我确更丑了。我懂了，下次不能让她来拍，妈的。

# 键盘

呜呜～之前在毕业宿舍捡垃圾时捞到的键盘，陪伴我这么久，最近出了一点小毛病，所以我买了一个新的键盘，紫粉黑搭配，冰淇淋轴，哇靠，爽爆了，这不得在游戏里爆杀，这不得提高我的学习效率，三模连接，学习时桌面放书，键盘放抽屉，绝配！没有线的感觉太棒了，而且手感一流，还可以自定义灯光，爽了啦。希望它能够陪伴我更久～～～写这篇博客时用这个键盘简直是一种享受。最近开学了，大个子今天也去了学校，我叫她给我看看她的美女室友，真狗东西，不给我看，气屎我了，不写了，掰掰。
